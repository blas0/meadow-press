# RULES.md

**Appending new deployments:**
> Whenever the user specifies for a new project/deployment OR users prompt doesn't align with any exisiting deployments.

1. Create folder using a place + noun pattern

- eg. lakebed, meadowpress, rivertalk, mountainjump

2. Effective intialization of lakebed web apps can be referenced in `DOCS.md`

3. Always deploy private/password-protected lakebed web apps, providing the link + pass to the user after deployment.

**Modifying existing deployments:**

Whenever the user specifies a change for a specified deployment OR users prompt aligns with an existing deployment, if alignment covers multiple deployments – `AskUserQuestion` for clarity.
